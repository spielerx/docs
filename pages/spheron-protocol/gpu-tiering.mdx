We support almost all Nvidia GPUs and are continuously adding support for new GPUs, including AMD GPUs. This table is useful if you want to lease a GPU for deployment or become a provider and lend your GPU. Here is the definition of the columns:

- **GPU Short Name:** This is useful for users who want to lease a GPU. When specifying the GPU you want to lease, you need to provide the GPU short name instead of the full name. Otherwise, the network wonâ€™t recognize the GPU type, and the deployment will fail.
- **Tier:** This helps providers understand which resource tier and associated multiplier they are eligible for. This is important for receiving rewards as part of the provider network.
- **Provider Capability:** This is crucial for providers when adding their provider information on-chain. Providers need to pass the correct capabilities based on the GPUs they are offering to the network. The capabilities must match what we have provided here. If incorrect capabilities are submitted, it can result in reward slashing.
